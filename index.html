01.

About Me

Hello! My name is Ashmeet and I enjoy creating websites and projects that live on the internet. My interest in front-end web development started when I realised that while I love the coding part of a project, I also enjoy dabbling in the designing aspect of it. So I decided to give web development a go and haven't looked back since.

Currently I am working as a software engineer in TCSBancs application product of Tata Consultance Services where I handle the UI part of the notification module.

Here are some of my skills : HTML CSS Javascript Java React Git
